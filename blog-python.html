# 탄탄한 기본기를 위한 다형성(Polyglot) 학습

백엔드 시스템은 상황에 맞는 최적의 도구를 선택하는 것이 중요합니다. 단순히 하나의 언어 문법을 외우는 것을 넘어, Java의 객체지향 설계(OOP)와 추상화, Python의 데이터 파싱과 API 연동, 그리고 C언어의 메모리 구조를 종합적으로 탐구하며 서버 엔지니어로서의 기본기를 다지고 있습니다.

# 무엇을 기록하고 고민하는가?

단순한 튜토리얼 따라 치기가 아닌, 실제 코드를 설계하며 겪는 트러블슈팅과 아키텍처적 고민을 구글 기술 블로그에 기록하고 있습니다.

  • Java: 명품 자바 에센셜을 기반으로 클래스와 객체, 상속, 캡슐화를 활용한 데이터 무결성 보장
  • Python: Try-Except를 활용한 방어적 프로그래밍 및 제너레이터(Generator)를 통한 메모리 최적화
  • Architecture: 객체 지향 원칙(SOLID)과 효율적인 관계형 DB 설계 등

블로그 기록 예시: 객체지향 캡슐화와 생성자 위임

데이터를 보호하기 위해 접근 제어자를 설정하고, this()를 통해 중복 코드를 제거하는 로직 고민의 흔적입니다.

public class UserProfile {
    // 외부 접근을 차단하여 데이터 무결성 보장 (캡슐화)
    private String username;
    private int level;

    // this()를 활용한 생성자 위임 (중복 최소화)
    public UserProfile() {
        this("Guest", 1); 
    }

    public UserProfile(String username, int level) {
        this.username = username;
        this.level = level;
    }
}
