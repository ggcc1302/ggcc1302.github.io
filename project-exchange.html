Term Project / 1인 개발

실시간 API 연동
환율 변환 프로그램

1. 프로젝트 개요

Python 기초 문법과 외부 데이터 통신(Network) 역량을 다지기 위해 진행한 전공 텀 프로젝트입니다. 한국수출입은행 등 공공데이터 API를 활용하여 전 세계 주요 통화의 실시간 환율 정보를 수집하고, 사용자 친화적인 GUI 환경에서 즉시 계산해 주는 데스크톱 애플리케이션을 1인 기획 및 개발하였습니다.

2. 핵심 개발 주안점

  • REST API 및 JSON 파싱: requests 라이브러리를 통해 외부 환율 API 서버와 통신하고, 응답받은 JSON 형태의 데이터를 파이썬 딕셔너리로 파싱하여 필요한 환율 수치만 정확하게 추출하는 로직 구현
  • 직관적인 UI/UX 구현: Tkinter를 활용하여 사용자가 쉽게 국가를 선택하고 금액을 입력할 수 있는 데스크톱 윈도우 창 및 콤보박스 인터페이스 설계
  • 데이터 무결성 확보: 실시간 통신 시 발생할 수 있는 데이터 누락을 방지하고 정확한 소수점 계산을 처리하는 알고리즘 적용

3. 트러블슈팅 및 예외 처리 (Troubleshooting) 💡

이슈: 네트워크 지연 및 사용자 입력 오류로 인한 프로그램 강제 종료

발생 상황: API 서버 점검으로 인해 응답이 늦어지거나, 사용자가 금액 입력란에 숫자 대신 문자를 입력할 경우 런타임 에러가 발생하며 프로그램이 튕기는 현상 발생.

해결 과정: 프로그램의 안정성을 최우선으로 고려하여 철저한 예외 처리(Exception Handling)를 도입했습니다. try-except 블록을 세분화하여 구성했습니다.

1) 통신 오류: API 타임아웃 발생 시 except requests.exceptions.RequestException으로 잡아 "네트워크 연결을 확인하세요"라는 안내 팝업을 띄우도록 처리
2) 입력 오류: 문자 입력 시 ValueError를 캐치하여 데이터 연산 전에 입력을 초기화하도록 방어 로직(Defensive Programming) 구현
